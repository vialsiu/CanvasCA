## Features
(Left Toolbar)
- Rotate Text
- Rotate Image
- Adjust Brightness
- Apply filters
- Adjust RGB
  
(Right Toolbar)
- Insert Image
- Draw Tool
- Eraser Tool
- Add Text
- Download File
- Delete item
- Adjust size (draw/erase)
- Select color (draw tool)
- Select color (erase tool)
- Apply masks
- Apply convolutions
- Choose font style
